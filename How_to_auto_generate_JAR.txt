「Lambda_s3 auto generate JAR file user manual」

Windows version :

1.Click the ”Lambda_s3_makeJar.bat” just in one button,
then .bat file will help you to generate Jar file automatically.

2.Then open path ../build/libs ,you can see the Jar file already 
generated.


Mac Os :
1.Open Terminal and input ./Lambda_s3_makeJar.sh


「Lambda_s3 自動生成JAR檔說明」

微軟：

步驟1.
一鍵點擊 「Lambda_s3_makeJar.bat」檔案
則會自動產生Jar file.

步驟2.
打開自動產生的目錄 ../build/libs，則可以看到自動產生的JAR檔案

蘋果用戶：
打開終端機，之後輸入  ./Lambda_s3_makeJar.sh 即可
